Basic Git Commands

Command Description
git init Initialize a new Git repository.
git clone <repo-url> Clone an existing repository from a remote server.
git status Check the status of files in the working directory.
git add <file> Add a file to the staging area.
git commit -m "message" Commit changes with a message.
git push origin <branch> Push committed changes to a remote branch.
git pull origin <branch> Fetch and merge changes from the remote branch.
git branch List all branches.
git checkout <branch> Switch to another branch.
git merge <branch> Merge another branch into the current branch.
git log Show commit history.
git reset --hard HEAD~1 Undo the last commit.
